Diagnóstico e Tratamento

Baciloscopia (TRM-TB)

Registrar:

  • SR para detectável sensível à rifampicina.
  • RR para detectável resistente à rifampicina.
  • DT para detectável traços (resistência não avaliada)
  • DI para detectável indeterminado à rifampicina.
  • ND para não detectável.
  • IE para Inválido/erro/sem resultado.
  • MTBpara MTB detectado - Traços.
  • NR para não realizado.

PPD

Registrar:

  • Não Reagentequando o sistema imune não reagiu à substância. Indica que a pessoa não teve contato com a bactéria ou, em casos raros, que a imunidade está muito baixa (falso-negativo).
  • Reagente Fracoquando houve contato prévio com a bactéria da tuberculose, com micobactérias ambientais, ou é uma resposta à vacina BCG. É considerado positivo em grupos de risco (como pessoas com HIV ou que convivem com alguém doente).
  • Reagente Fortequando há uma forte probabilidade de infecção latente pela bactéria da tuberculose.

Radiografia do tórax

Registrar:

  • N para aspecto normal.
  • Sug para imagem sugestiva de TB.
  • Seq para sequela, quando houver imagem sugestiva de lesão cicatricial.
  • OP para imagem sugestiva de outra patologia não tuberculosa.
  • NR para não realizada.

Teste HIV

Registrar:

  • Positivopara teste rápido positivo.
  • Negativopara teste rápido negativo.

Esquema de Tratamento

Registrar:

  • CN para caso novo.
  • RIT para reingresso após interrupção do tratamento (abandono).
  • RME para reentrada após mudança de esquema.
  • RE para reexposição.
  • RS para reentrada após suspensão por condição clínica desfavorável ao tratamento.

TARV

Registrar:

  • S para sim.
  • N para não.
  • IGN para ignorado.
